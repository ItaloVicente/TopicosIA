DESCRIPTION:

Build script and main for Indexer

Build script can be used to build both Indexer and Projector using

./build.sh indexer
./build.sh projector
./build.sh            (builds everything)
./build.sh clean      (clean up)

By default, it will build both. Binaries will be placed under bin.
