DESCRIPTION:

Remove MAINTAINERS.md file from repository root directory

The idea of MAINTAINERS.md file was to indicate to contributors which
component is maintained by which committer and to know who to ask for
a review. However, in practice this didn't play out as expected. Instead
each committer is able to review any of the contribution.

If a committer is less familiar with a component, that committer can
request a review by another committer.

Moreover, the MAINTAINERS.md is outdated and the referenced list of
committers is not up-to-date.

Contributors can find the list of active committers here:
https://projects.eclipse.org/projects/tools.tracecompass/who
