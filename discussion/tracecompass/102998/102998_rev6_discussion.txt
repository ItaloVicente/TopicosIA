counters.ui: improved filtered checkbox tree viewer behavior

When a trace is loading, the tree's root entry is displayed in the
Counters view. The user can toggle this root entry. If the root entry is
checked when the trace is done loading, the entire tree will be checked
and the chart will display all data series.
