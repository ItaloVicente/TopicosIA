tmf: Add Xml Manager view

This view allows the user to manage the XML analysis that he imports.
With this view, he can import and remove XML analysises, and
also edit them. The edit functionality simply opens a view that shows
the nodes present in the XML file and their attributes. The user can
modify these attributes and save these changes.
