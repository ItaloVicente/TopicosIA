add few useful mods to the spinner

delayed-spinner:
  Inner spinner will be shown with delay if mn-spinner
  has this class, meanwhile spinner background will be
  shown without delay

expanded-spinner:
  this class should be used to overlap elements shadow
  that goes beyond the spinner

fixed-spinner:
  is used in order to fix spinner position
