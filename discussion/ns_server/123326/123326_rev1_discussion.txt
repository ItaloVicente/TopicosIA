MB-38207 Enclose Referer and User-agent in quotes

The change affects only some log entries in the http_access_internal.log file
(entries in the http_access.log file are unchanged to maintain
compatibility).

For log entries generated by "internal" users ("@<user>") the "Referer"
and "User-agent" fields are now wrapped in double quotes.

127.0.0.1 - @cbq-engine-cbauth [04/Mar/2020:11:48:07 -0800] \
        "GET /pools/default/buckets/default/ddocs HTTP/1.1" 200 11 \
         - "Go-http-client/1.1"
           ^                  ^
           +----- Quotes -----+

https://httpd.apache.org/docs/2.4/mod/mod_log_config.html#logformat
does this as well as the examples I found on the web.
