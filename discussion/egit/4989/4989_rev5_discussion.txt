Remove Gerrit Configuration Page from Clone Wizard

Now we have the possibility to plug in clone sources. The Gerrit
configuration will be provided by the appropriate Gerrit extension.

another source or using a custom URI, he can still configure the
remote as Gerrit in the context menu of the Repositories View.
