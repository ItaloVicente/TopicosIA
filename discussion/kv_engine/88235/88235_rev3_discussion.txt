MB-27063 Fix AuditFilteringTest test

Fix test that incorrectly waits for the string "uuid" to appear in the
audit log.  It should be waiting for an event generated by the user
"another".

Also increase the maximum wait time from 5 seconds to 10 seconds.  This
ensures the test passes on slow disk systems where it takes a long time
for the data to appear in the audit log.
