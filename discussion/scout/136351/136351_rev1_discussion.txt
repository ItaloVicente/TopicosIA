DESCRIPTION:

Add property to configure trusted certificates

Trusted certificates can be stored in remote-files/certificates. But
sometimes it is desirable to add certificates to the trust store from a
classpath or file path.
For this a new list property 'scout.trustedCertificates' has been
introduced.
