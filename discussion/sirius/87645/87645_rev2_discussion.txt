[510017] EEF Specifier Properties: widgets for Group domain class

On the specifier side, on a Group, add the possibility to automatically
generates appropriate widgets (with all required fields filled) for all
structural features of a domain class.
It is also possible to generate the appropriate widget for one specific
structural feature of the domain class associated to the Group.
These actions are only available if the domain class is valid.
The specifier should also fill the metamodels field from the
"Properties" model element.
