[DRAFT] Add reset filters command

This commit is a draft of adding an action in the dropdown filters menu
(in the tabbar) to reset the filters to their initial state. The initial
state corresponds to the default Concern defines in the VSM. If there is
not a default Concern, this action is not displayed.
The action does nothing in this commit. It is just a UI POC.
